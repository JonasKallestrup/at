atdisplay

PURPOSE

ATDISPLAY checks the verbosity level in the global variable GLOBVAL

SYNOPSIS

function atdisplay(verbosity,message)

DESCRIPTION 

ATDISPLAY checks the verbosity level in the global variable GLOBVAL
          and displays message if this is greater than the verbosity
          for this message.

  See also NUMDIFPARAMS
